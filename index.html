Establecer y Finalizar Sesión Remota en PowerShell

Para establecer una sesión remota en PowerShell, puedes usar el siguiente código:

$session = New-PSSession -ComputerName NombreDelEquipo -Credential (Get-Credential)

Este código se utiliza para conectarse a un equipo remoto y ejecutar comandos de PowerShell en él.

Para finalizar la sesión remota, puedes usar el siguiente código:

Remove-PSSession $session

Este código cierra la sesión remota previamente establecida.
